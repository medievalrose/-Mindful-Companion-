# -Mindful-Companion-

心理咨询候诊期AI陪伴工具 —— 温暖的AI倾听者，基于CBT理论的智能情绪支持伙伴

## 项目简介

中国心理咨询资源严重不足，患者从预约到首次面询平均等待2-6周。等待期内，患者往往处于情绪最脆弱的阶段，却缺乏任何专业支持。Mindful Companion 正是为填补这段"空白期"而设计。

本应用提供**情绪日记**、**CBT自助练习**、**危机预警转介**三大核心功能，帮助用户在候诊期间获得结构化的情绪支持，同时建立危机识别与人工转介机制，守住安全底线。

## 核心功能

### 📝 情绪日记
- 自由书写今日心情，AI 给出温暖共情回应
- 自动识别情绪类型与强度（焦虑/低落/愤怒/孤独等）
- 7天情绪趋势可视化，直观感受情绪变化

### 🔄 CBT练习室
- **认知重构**：识别消极自动思维，寻找替代解释
- **行为激活**：规划微小积极行动，打破负面循环
- **焦虑记录表**：记录担忧、评估概率、追踪实际结果

### 🆘 危机预警
- 实时检测自伤/自杀相关表述
- 触发醒目预警弹窗，引导拨打专业求助热线
- 内置全国心理援助热线信息

## 技术栈

- **前端**：HTML / CSS / JavaScript（单页应用）
- **AI模型**：百度千帆
- **数据可视化**：Chart.js
- **本地存储**：localStorage
- **开发工具**：Trae
